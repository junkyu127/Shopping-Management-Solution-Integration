# 🛒 쇼핑몰 통합 관리 솔루션 API 연동 및 유지보수

2년 동안 쇼핑몰 통합 관리 솔루션 **'이지오토'**의 백엔드 개발 및 유지보수를 담당하며, 다양한 이커머스 플랫폼의 데이터를 통합 관리하는 업무를 수행했습니다.

## 🛠 사용 기술
- **Language:** PHP, C#
- **Database:** MySQL
- **Version Control:** Git
- **Communication:** REST API (JSON/XML)

## 🚀 주요 업무 및 성과

### 1. 쇼핑몰 Open API 연동 및 풀 사이클(Full-Cycle) 관리
- 국내외 주요 쇼핑몰(네이버 스마트스토어, 쿠팡 등)의 API를 활용하여 이커머스 핵심 운영 프로세스 자동화.
- **주문 관리**: 실시간 주문 확인 및 자동 주문 수집 로직 구현.
- **배송 관리**: 솔루션 내 송장 번호 입력 시 각 쇼핑몰 API를 통한 실시간 송장 정보 전송.
- **CS 및 취소 연동**: 쇼핑몰 측의 주문 취소 상태를 실시간 동기화하고, 게시판 API 연동을 통해 통합 고객 응대 시스템 구축.
- 각 플랫폼별 상이한 데이터 규격을 솔루션 내부 표준 포맷으로 가공하는 파이프라인 관리.

### 2. 업체별 맞춤형 비즈니스 로직 설계 (Hard-coding 관리)
- 특정 업체의 개별 요구사항에 대응하기 위한 **전략적 분기 처리** 로직 구현.
- 하드코딩 요청 시 시스템 전체의 안정성을 해치지 않도록 코드 독립성을 확보하고, 유지보수를 고려한 로직 분리 수행.

### 3. 데이터 정합성 유지 및 트러블슈팅
- API 호출 제한(Rate Limit) 및 타임아웃 발생 시 재시도(Retry) 로직을 통해 데이터 유실 방지.
- MySQL 쿼리 최적화를 통해 웹 및 프로그램 내 실시간 데이터 반영 속도 개선.

## 📝 업무 경험 요약
- 보안 정책상 실무 코드를 직접 업로드할 수 없으나, 본 레포지토리는 해당 기간 동안 경험한 기술적 고민과 해결 과정을 기록하기 위해 생성되었습니다.
- API 연동 중 발생하는 예외 상황 처리 및 대용량 데이터 가공에 강점이 있습니다.
